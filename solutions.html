Pipelining is a technique where multiple instructions are overlapped during execution.
Cache memory is a small, fast memory between CPU and RAM to speed up data access.
RISC uses a small set of simple instructions; CISC uses complex instructions.
A list is an ordered collection of items in Python.
Functions are reusable blocks of code that perform a specific task.
A dictionary is a collection of key-value pairs in Python.
A process is an independent program in execution with its own memory, while a thread is a lightweight part of a process that shares the same memory.
Deadlock is a condition where processes are stuck waiting for resources held by each other; it can be prevented by avoiding circular wait,using resources ordering, or applying safe state algorithms like Banker's algorithm.
Virtual memory is a technique where secondary storage is used as an extention of RAM, allowing programs to run even when physical memory is limited.
Human values are principle like respect, trust, and care that behavior towards living harmoniously with oneself and others.
Harmony in the self means peace and clarity within thoughts and feelings, while harmony in the family is about building trust, respect, and love among family members.
Needs of the body are material and temporary like food and shelter, wherease needs of the self are non-physical and continuous like happiness and respect.
An algorithm is a step-by-step method to solve a problem; a good algorithm is correct, rfficient, finite,clear, and has defined input and output.
Time complexity measures how the running time of an algorithm increases with input size, and space complexity measures the amount of memory used.
Divide and conquore is a strategy where a problem is broken into smaller subproblems, solved individually, and then combined, as seen in algorithms like Merge Sort.
